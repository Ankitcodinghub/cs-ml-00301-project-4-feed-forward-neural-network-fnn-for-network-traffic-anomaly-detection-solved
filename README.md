In this project, we consider NSL-KDD dataset D = (A,N) that contains attack sub-dataset A and normal sub-dataset N. In this project, we choose D = KDDTrain + .txt in the NSL-KDD dataset for training and D = KDDT est + .txt in the NSL-KDD dataset for testing. Now, you picked 4 attack types, e.g., A1 − A4 ⊂ A along with normal data N, you start with a subset A′ ⊂ A for training the model along with all the normal data N. The four classes of attacks include: A1: Denial of Service (DoS), A2: Probe, A3:User to Root(U2R), and A4: Remote to Local (R2L). Refer to lab CS-ML-00101 for more details of these four classes of attacks their their included sub-classes.
